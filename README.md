
# 1D Array Student Scores Analyzer

A simple Python console based program for recording, managing, and analyzing student scores.

Author: Jhames Rhonnielle T. Martin 
Course: BSIS 1A
Github: jrmph

---

## Project Description

This program is a menu optional Python application that allows the user to manage a list of student scores. It provides basic operations such as viewing scores, counting passing students, finding the highest and lowest values, searching for a score, inserting new data, and deleting existing data. The program also includes simple loading animations and a formatted console interface for better user experience.

---

## Features

- Input multiple student scores
- Display all recorded scores
- Count students who passed with a score of 75 or higher
- Display the highest and lowest score
- Search for a specific score
- Insert a new score at a chosen position
- Delete a selected score
- Display the final updated list before exiting

---

## How to Run the Program

1. Install Python 3 on your computer.
2. Save the program as a Python file (for example: `main.py`).
3. Open a terminal or command prompt in the file location.
4. Run the program using:

python main.py

5. Follow the instructions on the screen.

---

## How the Program Works

1. The program starts by asking for the number of students.
2. The user enters each student's score one by one.
3. After all scores are entered, the main menu is displayed.
4. The user selects an option from the menu to perform different operations.
5. The program continues running until the user chooses the exit option.

---

## Main Menu Options

1. Display all scores  
2. Count students who passed (75 and above)  
3. Show highest and lowest score  
4. Search for a score  
5. Insert new score  
6. Delete a score  
7. Show final list and exit
